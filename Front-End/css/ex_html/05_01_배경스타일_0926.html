배경 스타일

background-color : 요소의 배경색을 지정하는 속성

선택자 : {background-color : 색상명 | rgb | rgba | 16진수 } 과 같이 사용. 배경 색상을 지정하지 않은 요소는 흰색처럼 보이지만, 투명한 상태(body에 색상을 지정해보면 확인 가능)

background-clip : 배경색의 적용 범위를 지정하는 속성

선택자 : {background-clip : border-box | padding-box | content-box}와 같이 사용

background-image : 요소의 배경에 이미지를 적용

선택자 { background-image : url(이미지 경로) } 와 같이 사용

background-repeat : 배경이미지 반복 처리 속성

선택자 : { background-repeat : repeat | repeat-x | repeat-y | no-repeat }과 같이 사용

background-position : 배경 이미지의 위치를 지정하는 속성

선택자 : { background-position : x축위치 y축위치 }와 같이 사용

background-size : 배경 이미지 크기 조정 시 사용

선택자 : {background-size : auto | contain | cover | 크기 지정 }과 같이 서용

background-origin : 배경이미지 배치 기준

선택자 : {background-origin : border-box | padding-box | content-box }와 같이 사용

background-attachment : 배경 이미지 고정 여부를 결정하는 속성

선택자 : { background-attachment : scroll | fixed }
