width / height 속성 : 컨텐츠가 차지하고 있는 영역의 크기를 지정할 때 사용

선택자 {width : 크기(단위); }
선택자 {height : 크기(단위); } 와 같이 사용

인라인 요소. 내용의 합 만큼 높이와 너비가 지정되고, 스타일 직접 지정하는건 불가능 여러개를 줄을 바꾸어 작성해도 한 줄에 표기.
블록 요소. 가로는 기본적으로 한 줄 전체를 차지한다. 높이는 높이를 지정한 만큼 차지하거나, 지정하지 않았다면 내용의 합 만큼 높이를 차지한다.









자식 Div - 넓이 미지정 시, 부모 요소의 넓이만큼 - 높이 미지정 시, 내부 컨텐츠 높이만큼 - 부모 요소보다 넓이 or 높이를 크게 지정하면, 지정한만큼 영역을 차지 - 부모 요소의 크기를 기준으로 넓이 or 높이를 지정할 수 있다. (%)

display : 요소의 화면 배치 방법

선택자 : {display : none | block | inline | inline-blck | flex} none : 화면 숨김 처리
block : 인라인 요소 -> 블록 요소
inline : 블록 요소 -> 인라인 요소
inline-block : 인라인 요소의 특징과 블록 요소의 특징 모두 (한 줄에 여러 요소가 올 수 있고, 그기 지정 가능)
flex : 요소를 유연하게 배치
