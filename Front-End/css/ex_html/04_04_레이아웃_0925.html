z-index 속성 : 요소들이 쌓이는 순서를 지정할 때 사용

선택자 { z-index : 숫자}와 같이 사용

visibility 속성 : 화면에 요소가 보이거나 보이지 않게 하는 속성

선택자 { visibility : visible | hidden }과 같이 사용.

opacity 속성 : 요소의 투명도를 지정할 때 사용

선택자 {opacity : 0 ~ 1}과 같이 사용.

float 속성 : 요소의 배치를 왼쪽 또는 오른쪽으로 배치하기 위해 사용하는 속성

선택자 {float : left | right | none}
