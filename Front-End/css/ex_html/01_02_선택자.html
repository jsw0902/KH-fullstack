기본 속성 선택자

기본 선택자 뒤에 []를 사용하여, 속성명칭과 속성값을 작성한다.
선택자[속성 비교연산 속성값]{스타일명: 값;} 형식으로 사용
비교 연산
-A : 속성값이 A인 것.
^=A : 속성값이 A로 시작하는 것
$=A : 속성값이 A로 끝나는 것.
*=A : 속성값에 A가 포함되는 것.

자손 선택자와 후손 선택자

자손 선택자 : 부모 요소 바로 하위의 요소.
후손 선택자 : 부모 요소 하위 요소 모두.
선택자>자손선택자 {스타일명 : 값;}
선택자 {후손선택자 스타일명 : 값}
모든 자손 요소들은 "후손" 요소라고 말할 수 있다.
태그에 id, name, class를 지정하지 않고 관계만으로 선택할 때 사용

동위 선택자

동위 관계에서 뒤에 위치한 태그를 선택할 때 사용한다.
선택자a+선택자b {스타일명:값;} : 바로 뒤의 요소를 선택할 때
선택자a~선택자b {스타일명:값;} : 뒤에 있는 모든 요소를 선택할 때

반응 선택자

마우스의 이벤트에 따라서 달라지는 선택자.
active : 마우스로 태그 영역을 클릭했을 때 스타일 적용.
선택자 : active{스타일명 : 값;} 으로 사용

hover : 마우스가 태그 영역 위에 위치했을 때 스타일 적용.
선택자 ㅣ hover {스타일명:값;} 으로 사용
단, 마우스가 영역 밖으로 벗어나면 스타일 복구.

상태 선택자

태그 상태에 따라서 선택할 때 사용.
input 태그 : checked | read-only | disavled {스타일명: 값;}
checked : 체크박스나 라디오 버튼을 체크한 상태
read-only : input 태그가 읽기 전용일 때
disavled : input 태그가 읽기 전용이고, 서버로 전송되지 않는 상태
