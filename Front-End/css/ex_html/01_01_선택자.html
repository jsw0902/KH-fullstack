CSS 선택자

선택자 : 특정한 HTML 태그를 선택하는 것. 선택하여 원하는 스타일과 기능을 적용할 수 있다.

전체 선택자

* {스타일 명: 값;} HTML 문서 내의 모든 태그를 선택할 때 전체 선택자를 사용할 수 있다.

태그 선택자

태그명칭 {스타일명 : 값;} HTML 문서 내의 동일한 태그를 선택할 때 사용한다.

아이디 선택자

#아이디 {스타일명 : 값;} HTML 파일 내에 해당하는 아이디 속성의 값을 가진 태그를 선택할 때 사용한다. 동일한 HTML 파일 내에, 동일한 ID 값을 가지는 태그를 여러개 작성하지 말 것.

클래스 선택자

.클래스명 {스타일명 : 값;} HTML 문서 내에 여러개의 태그를 동일한 클래스명으로 지정하여, 선택할 때 사용한다. 동일한 스타일을 여러 태그에 적용하고 싶을 때 태그의 class 속성에 동일한 명칭을 작성해준다.
