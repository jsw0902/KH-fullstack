box-sizing

box-sizing : 컨텐츠 영역의 width와 height의 범위를 지정하는 속성

선택자 { box-sizing : content-box | border-box }와 같이 지정

box-sizing : content-box인 경우.
box-sizing : border-box인 경우. 왼쪽 border(10)+왼쪽 padding(50)+ 가로 길이(80)+오른쪽 padding(50)+ 오른쪽 border(10) = 200px
















position : 페이지 요소들의 위치를 설정하는 속성

선택자 : (position : relative | absolute | fixed | static[기본값])과 같이 사용.
