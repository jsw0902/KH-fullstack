테두리 스타일 지정



테두리의 스타일(종류) 지정

선택자 {border-[방향(생략가능)]-style : none | hidden | dashed | dotted | double | groove | inset | outset | ridge | solid } 와 같이 사용

테두리 두께 지정

선택자 {border-[방향(생략가능)]-width : thin | medium | thick | px } 와 같이 사용

테두리의 두께 지정

선택자 {border-[방향(생략가능)]-color : 생상값 } 과 같이 사용

테두리 스타일을 한번에 지정

선택자 {border-[방향(생략가능)] : width값 style값 color값 }과 같이 사용

박스의 모서리 영역을 둥글게 하는 속성

선택자 {border-[방향(생략가능)]-radius : 숫자}

박스의 그림자를 지정하는 속성

선택자 {box-shadow : 수평거리 수직거리 흐림 번짐 색상} 과 같이 사용.


여택 스타일

margin : 요소(Element)와 요소(Element)간의 여백

선택자 {magin-[위치(생략가능)] : 거리 } -> 위치는 top, bottom, left, right
선택자 {margin : 거리} 4가지 방향에 대해서 한번에 설정
선택자 {margin : 거리1 거리2 } -> 거리1 : top과 bottom에 대한 거리, 거리2 : left와 right에 대한 거리
선택자 {margin : 거리1 거리2 거리3 거리4} -> 거리1 : top, 거리2 : right, 거리3 : bottom, 거리4 : left (top을 시작으로 시계방향)

padding : 요소 (Element)의 테두리(border)와 내부 컨텐츠(content)사이의 여백

설정 방법은 margin과 동일

margin test1
margin test2
margin test3


padding test1
padding test2
padding test3
