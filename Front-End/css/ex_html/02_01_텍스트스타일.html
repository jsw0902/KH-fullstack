font-size 속성 : 텍스트의 크기를 지정할 때 사용

font-style 속성 : 텍스트의 기울임을 지정할 때 사용

선택자{font-style : normal | italic | oblique;}과 같이 사용

  • italic 기울임 글꼴
  • oblique 기울임 글꼴

font 속성 : 글꼴 관련 스타일을 한번에 지정할 떄 사용

선택자 {font : font-style font-variant font-weight font-size/line-height font-family}과 같이 사용

  • 글꼴 관련 스타일 테스트1(16px 궁서체)
  • 글꼴 관련 스타일 테스트2(italic bold 20px 고딕체)
