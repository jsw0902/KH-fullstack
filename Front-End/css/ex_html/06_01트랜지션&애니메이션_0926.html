transform : 페이지에서 요소를 변형을 시키기위한 기능

1. 지정한만큼 좌/우 이동

2. 지정한만큼 상/하 이동

3. 지정한만큼 상하/좌우 이동

4. 지정한만큼 좌우 확대

5. 지정한만큼 상하 확대

6. 지정한만큼 상하,좌우 확대

7. 지정한 각도만큼 회전

8. 지정한 각도만큼 요소를 기울이기(좌우)

9. 지정한 각도만큼 요소를 기울이기(상하)

transition : 요소의 스타일 변화를 애니메이션처럼 변경

1. 시간에 따른 변화

2. 시간 지연




애니메이션

트렌지션과 마찬가지로, 중간단계를 지정하여 자연스럽게 연결시키는 기능
