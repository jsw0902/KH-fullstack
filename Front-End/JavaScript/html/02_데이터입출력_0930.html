2. 데이터를 입력받는 구문

2.1) [window.]confirm("질문 작성");

2.1) [window.]confirm("질문 작성");

confirm 함수 호출 시, "질문 내용"과 "확인/취소" 버튼이 존재하는 알림창 발생
"확인" 버튼 클릭 시, true를 반환하고 "취소" 버튼 클릭 시, false를 반환.


2.2) [window.]prompt("질문 작성");

prompt 함수 호출 시, "질문 내용"을 작성하면, 알림창에 질문 내용이 보여지고,
입력할 수 있는 텍스트 상자와 "확인/취소" 버튼이 생성된다.
확인 버튼을 누르면 텍스트 상자에 입력된 값이 문자열로 반환된다.(내부에 숫자를 입력해도 문자열로 반환)
취소 버튼을 누르면 null을 반환. (취소 버튼을 누르면 텍스트 상자에 값이 입력되어 있어도 null를 반환)
